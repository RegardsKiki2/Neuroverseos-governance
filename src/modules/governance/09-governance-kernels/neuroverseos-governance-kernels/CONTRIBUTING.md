# Contributing

## What This Repository Accepts

This repository accepts:

- Bug reports related to artifact formatting or metadata accuracy
- Documentation corrections (typos, broken links, factual errors)
- Requests to clarify governance scope or boundary definitions

## What This Repository Does Not Accept

This repository does not accept:

- New governance artifacts or templates
- Modifications to existing enforcement logic
- Code contributions of any kind
- Generation logic, compilers, abstraction helpers, or tooling
- Pull requests that introduce executable dependencies

All governance artifacts are compiled from source modules that are not distributed in this repository. There is no mechanism to contribute new artifacts, extend enforcement logic, or modify compiled outputs from this repository.

## How to Report an Issue

Open a GitHub Issue with:

- A clear description of the problem
- The specific file and line affected (if applicable)
- Expected vs. actual behavior

## Rejection Policy

Any pull request that introduces generation logic, compilation tools, abstraction helpers, or executable code will be closed without review. This is not a development repository. It is a publication surface for compiled artifacts.
